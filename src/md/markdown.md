# 마크다운 문법

## 마크다운이란?

마크다운은 일반 텍스트 문서에 서식 요소를 추가하는데 사용할 수 있는 가벼운 마크업 언어 2004년 존 그루버에 의해 만들어 졌으며, 쉽게 쓰고 읽으며 HTML로 변환이가능하다. 특수 기호와 문자를 이용한 간단한 구조의 문법을 사용해서 웹에서도 빠르게 컨텐츠를 작성하고 직관적으로 인식할 수 있다.

---

## 제목

- 큰제목 :문서 제목

  ```
  이것은 큰제목 입니다
  ===================
  ```

  # 이것은 큰 제목입니다
  - 작은 :제목문서 부제목

  ```
  이것은 작은 제목입니다
  ---------------------
  ```

  ## 이것은 작은 제목입니다

- 또는 단어나 구 앞에 '#' 를 넣는다

```
# h1 제목
## h2 제목
### h3 제목
#### h4 제목
##### h5 제목
###### h6 제목
```

# h1 제목

## h2 제목

### h3 제목

#### h4 제목

##### h5 제목

###### h6 제목


# br 태그

-'< br >' 태그를 이용해서 줄바꿈을 할 수 있다. 엔터로 칸을 띄우면 다음행으로 넘어감.

```
마크다운을 위한 예제입니다
마크다운을 위한 <br> 예제입니다
```

마크다운을 위한 예제입니다
마크다운을 위한 <br> 예제입니다

#텍스트 강조

- 기울여 쓰기 :'\*'또는 '\_'로 감싼 텍스트
- 두껍게 쓰기: '\*\*'또는 '\_\_'로 감싼 텍스트
- 취소선 : '~~' 로 감싼텍스트
- 이텔릭체와 두껍게를 같이 사용 할 수 있다

```
_기울이기_
**굵게 쓰기**
~~취소 선 쓰기~~
_이텔릭과 **강조**를 같이 쓸 수 있다_
```

_기울이기_
**굵게 쓰기**
~~취소 선 쓰기~~
_이텔릭과 **강조**를 같이 쓸 수 있다_

# 코드 블럭

- 간단한 인라인 코드는 앞뒤로 '`'를 붙인다
- 또는 '\`\`\`' 혹은 '~~~'

# 체크리스트

- 줄 앞에 `-[x]`를 써서 완료된 리스트 표시
- 줄 앞에 `-[]`를 써서 미완료된 리스트 표시.
- 체크 안에서 강조외에 여러 기능을 사용할 수 있습니다.

```
-[x] 이건 완료된거
-[ ] 이건 미완료
```

-[x] 이건 완료된거 -[ ] 이건 미완료

# 테이블

- 헤더와 셀을 구분할 때 3개 이상의 하이픈(\-) 기호가 필요
- 헤더 셀을 구분 하면서 콜론(:) 기호로 셀(열/칸) 안에 내용을 정렬 할 수 있음

```
테이블 생성

헤더1|헤더2|헤더3|헤더4
---|---|---|---
셀1|셀2|셀3|셀4
셀5|셀6|셀7|셀8
셀9|셀10|셀11|셀12
```
